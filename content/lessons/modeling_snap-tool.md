#  {{title}}
{{description}}

The snap tool can be used at any time by pressing the control-key - CTRL in Windows/Linux and control (not command) in MacOS - while moving an object. That object will then snap to the nearest type of element as selected in the snap menu, e. g. to the nearest face if "snap to" is set to face.

When the snap button is active, moving an object will always snap it to the selected element.
