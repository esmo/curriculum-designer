## Additional Notes

Suggested shortcuts and interface focus:

- `G`, `R`, `S` for transform operations
- `Shift+A` to add mesh objects
- `X` to delete objects
- `A` to select all
- `Shift+LMB` for active-selection workflows
- `I` to insert keyframes
- Timeline controls for playback and interpolation review
- `Ctrl+L` -> `Link Materials` to reuse materials across objects
