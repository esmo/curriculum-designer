# Blender Rigging Foundation Level 1
## Creating Armatures

### Skills

**Creating Armatures**
- Creating single bone
- Structure (tip, body. root)
- Rename bones

**Editing Armatures**
- Edit mode
- Extruding new bones

**Constraining Geometry**
- Child of
- Armature

**Animating Armatures**
- Pose mode

### Interface / Keyboard

- shift+a: add menu (Armature)

- tab: edit mode
  - e: extrude new bone
  - g/r: move/rotate bone components


- ctrl+p constraint menu

### Ticklist

**Can**
- create bones
- edit bones
- rename bones
- extrude bones
- constraint "child of" and "armature"
- pose the bones

**Know**
- what bones are
- how to edit basic shape
- constraint to a geometry
- pose bones
